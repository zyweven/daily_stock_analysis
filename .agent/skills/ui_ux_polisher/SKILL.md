# UI/UX Polisher Skill

本技能致力于将平凡的 Web 界面打磨成具有“高级感”和“优秀交互体验”的艺术品。

## 核心守则

### 1. 视觉层级 (Visual Hierarchy)
- **对比与焦点**：通过字号、粗细和颜色建立清晰的信息层级。
- **留白艺术**：严控组件间距，使用合理的 Padding 和 Margin 营造呼吸感。

### 2. 现代审美 (Modern Aesthetics)
- **质感渲染**：优先使用渐变边框、毛玻璃效果 (Backdrop Filter) 和柔和阴影。
- **动效反馈**：为按钮和卡片添加平滑的 Hover 缩放或颜色过渡效果。

### 3. 响应式体验 (Responsive Design)
- **容器自适应**：确保页面在不同屏幕尺寸下均有良好的视觉平衡感。

## 审计工作流
当被要求“使用 UI 优化技能审计页面”时：
1. 识别样式中“廉价感”的设计（如纯黑/纯白背景、无衬线字体样式单一、间距过紧）。
2. 提供符合 CSS 现代标准的重构建议（如 CSS Variables, Flexbox/Grid 优化）。
