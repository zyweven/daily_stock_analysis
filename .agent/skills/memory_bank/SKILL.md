# Memory Bank Skill

本技能旨在通过结构化文档维护项目的长期上下文，使 AI 助手具有“持久记忆”。

## 核心守则

### 1. 记忆同步 (Memory Sync)
- **更新触发**：每完成一个重大功能或修复一个复杂 Bug 后，必须更新相应的知识库文档（如 `contexts/context.md` 或 `.agent/knowledge/`）。
- **结构化存储**：信息应按项目背景、技术架构、重大决策、待办事项等维度进行分类。

### 2. 上下文导向 (Context-Driven)
- **启动检查**：在开始新任务前，优先查阅 Memory Bank 了解当前项目的进度和约束。
- **决策追溯**：记录“为什么这样做”，而非仅仅“做了什么”，以便未来追溯。

## 审计工作流
当被要求“使用记忆库技能”时：
1. 检查当前目录下是否存在 `contexts/` 或 `.agent/knowledge/`。
2. 自动提取当前任务的关键信息并更新至项目的“长效记忆”中。
