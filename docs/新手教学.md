# 🎓 零基础新手教学：AI 股票分析快速上手

欢迎使用 **A股自选股智能分析系统**！本教程将带您从零开始，在 5 分钟内完成您的第一次“专家会诊”分析。

---

## 第一步：准备工作 (API Key)

要让 AI 为您工作，您需要一个“打工证”（API Key）。

1. **推荐使用 Gemini**: 访问 [Google AI Studio](https://aistudio.google.com/) 获取免费 Key。
2. **备选 DeepSeek**: 获取 DeepSeek 官方 API Key 或使用支持 OpenAI 格式的转发。

---

## 第二步：配置环境

1. 在项目根目录下找到 `.env` 文件。
2. 将您的 Key 填入对应位置：
   ```ini
   GEMINI_API_KEY=您的GeminiKey
   DEEPSEEK_API_KEY=您的DeepSeekKey (可选)
   ```
3. 设置您的自选股：
   ```ini
   STOCK_LIST=600519,300750
   ```

---

## 第三步：启动可视化界面

在电脑终端（PowerShell）中输入：
```powershell
.\.venv\Scripts\python main.py --webui-only
```
看到“Uvicorn running on http://127.0.0.1:8000”字样后，打开页面。

---

## 第四步：发起一次“专家会诊” (核心功能)

1. 点击左侧导航栏的 **“会诊”** 图标（类似小人的图标）。
2. 在输入框输入股票代码，例如：`600519`。
3. **勾选模型**: 勾选您配置过的模型（如 Gemini 和 DeepSeek）。
4. **开始会诊**: 点击按钮，等待约 1 分钟。

---

## 第五步：查看并理解分析结果

分析完成后，您将看到：
1. **共识评分**: 综合多位“专家”的打分（0-100）。
2. **决策摘要**: 告诉您为何买入或减仓。
3. **模型对比表**: 直观查看不同模型是否存在观点分歧。

---

## 🚩 注意事项
- **股市有风险**: AI 建议仅供参考，不构成投资建议。
- **网络稳定**: 建议配置全局代理，并确保在 `.env` 中正确填写。
- **报错排查**: 如果页面刷不出来，请参考 `docs/操作手册.md`。
